============================================================
   Synology Drive 旧电脑(Win7)安装 - 快速上手
============================================================

  这台旧电脑最好能上网。把整个文件夹解压到桌面即可。
  下面 3 步做完就装好了。


------------------------------------------------------------
  第 1 步   先关杀毒软件 (最重要, 别跳过!)
------------------------------------------------------------
  兼容层 VxKex 会被 360 / Windows Defender 当成病毒拦截,
  这是正常现象 (它要 hook 系统才能让新软件跑在 Win7 上),
  不关掉的话安装包会被删、装不上。

  · 360:  右下角小图标 -> 暂时退出;  或把本文件夹加信任
  · Defender: 设置 -> 更新和安全 -> Windows 安全中心
              -> 病毒和威胁防护 -> 管理设置 -> 关闭"实时保护"

  >>> 装完之后记得再打开回来 <<<


------------------------------------------------------------
  第 2 步   右键 [一键安装.cmd] -> 以管理员身份运行
------------------------------------------------------------
  跟着屏幕提示按回车即可。中途它会:

  (1) 下载 + 装 VxKex           -> 弹窗里一路点 Next / 下一步
  (2) 下载 + 装 Synology Drive  -> 一路点下一步
  (3) 自动弹出两个窗口:
        · VxKex 设置窗口
        · Synology 的 bin 文件夹
      在 VxKex 窗口点 [Add Program] / 添加程序,
      地址栏按 Ctrl+V 粘贴(路径已自动复制好), 回车进去,
      按 Ctrl+A 全选里面所有 .exe, 点 打开, 再保存。
  (4) 问是否开机自启 -> 输入 1 (推荐)
  (5) 问是否测试启动 -> 输入 Y


------------------------------------------------------------
  第 3 步   打开 Synology Drive, 登录你的 NAS
------------------------------------------------------------
  开始菜单找到 "Synology Drive Client" 打开,
  能进入登录界面并连上 NAS = 成功!


============================================================
   出问题时用这几个小工具 (都在同一文件夹, 双击即可)
============================================================
  · 打不开 / 报错缺少 xxx.dll
        -> 双击  诊断启动问题.cmd  (会告诉你哪一步没配好)
  · 想确认"开机自启"到底有没有生效
        -> 双击  检查开机自启动.cmd

============================================================
   常见卡点
============================================================
  ● 下载卡住 / 失败 (旧电脑、网络差很常见)
      在一台新电脑上下载这两个文件, 放进本文件夹的
      downloads 子目录, 再重跑 一键安装.cmd (会自动跳过下载):
        1) Synology Drive Client-4.0.3-17892-x86.exe
           https://archive.synology.com/download/Utility/SynologyDriveClient
        2) KexSetup_Release_ 开头的那个 exe (别下 Debug 版)
           https://github.com/YuZhouRen86/VxKex-NEXT/releases/latest

  ● 装好客户端却连不上 NAS
      不是本工具的问题, 是服务端要求:
      NAS 要装 "Synology Drive Server" 套件, 且系统 DSM 7.0 以上。

  ● 干净的 Win7 一堆签名/加载报错 (缺前置补丁)
      按顺序装, 每步之间重启:
        SP1 -> KB4490628 -> KB4474419 (+ KB2533623, KB2670838)
      微软更新目录搜 KB 号: https://www.catalog.update.microsoft.com/


  更详细的说明看同目录的  使用说明.txt
